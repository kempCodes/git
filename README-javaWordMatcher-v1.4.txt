Name: Word Matching
Version: 1.4
Free Use
Author: @kempCodes -- Kemper Lee
-----------------------------------------------------------------
**How to use this program:**
-----------------------------------------------------------------
        1. Enter a String value in the Input pane
            (represents a name)
            - numbers, spaces, and punctuation allowed
        2. Description for the chosen word is given 

        3. Enter a second String value in console
            - again, all characters allowed
        4. Continue entering Strings until:
            a. the Strings match
            b. 5 guesses are hit
        5. Enter a 'Y', 'y', or "yes" to continue playing
-----------------------------------------------------------------
**Expected outcomes**
-----------------------------------------------------------------
        1. Equal Strings
            - The game ends early
            - User stats are printed in the Console
        2. Not Equal Strings
            - User stats are printed in the Console
-----------------------------------------------------------------
